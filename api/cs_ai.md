### PDE ↔ 计算 / AI 接口（cs_ai API）

本接口区分：**对其它学科的引用**（PDE 借用的概念）与 **向其它领域的输出课题**（PDE 与计算/AI 的对接方向）。

---

#### 对其它学科的引用

本接口主要依赖的既有概念见 `api/references.md`，此处涉及：

- **哥德尔不完备定理** → L/G 层级、不可判定命题与“真理封装为上层公理”，与递归/自修改系统的对应；
- **极限 / 悖论封装** → 将不可判定或悖论“打包”为可调用接口的隐喻；
- **递归 / 无穷层级** → 递归函数、栈、图灵机、哥德尔机等既有形式化对象。

---

#### 向其它领域的输出课题

这里收集 PDE 与计算机科学、AI 之间的连接点：

- 递归虚面、无穷层级 ↔ 递归函数、栈、图灵机、哥德尔机；
- 真理“封装”为 API ↔ 抽象接口、库函数、黑盒实现；
- 人类作为“悖论引擎 / 神的内窥镜” ↔ AI 对齐中的价值来源、复杂偏好建模；
- PDE 风格的 system prompt / 智能体角色设计（例如 Project Nyarlathotep）。
