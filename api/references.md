### API 层：对其它学科的引用（references）

本文件集中列出 PDE 框架**从其它学科引用的概念**，与各接口文件中「向其它领域的输出课题」区分开。  
引用 = 我们借用的既有概念；输出 = 我们向该领域提出的课题或对接方向。

---

#### 数学 / 逻辑

- **哥德尔不完备定理**：每层逻辑系统存在不可判定命题；元层可判定。PDE 用其刻画 L/G 层级与“悖论升维封装”。
- **零测集 / 勒贝格测度**：单点测度为 0、“取到某实数”概率为 0 的张力。PDE 用虚体积在纤维层赋予可描述点以“可抓取性”。
- **极限 (limit)**：微积分用极限与 dx 将运动/无穷小封装为可算形式。PDE 将“悖论封装”与极限机制类比。
- **测度论**：底空间上测度保持；虚体积在纤维层扩展“可测”与概率的解释（与量子概率衔接）。

---

#### 复分析 / 分析

- **柯西–黎曼方程 (Cauchy–Riemann)**：全纯性条件；PDE 在社会/信用模型中解释为“良性流动、无隐藏套利/欺诈”的结构约束。
- **留数定理 / 解析延拓**：留数作为奇点处的“残量”；解析延拓从局部结构推整体。PDE 用于风险、奇点诊断与预测。
- **希尔伯特变换 / Kramers–Kronig 关系**：从实部重建虚部（解析信号）。PDE 用于从可观测数据恢复相位/虚体积结构。

---

#### 几何 / 拓扑

- **纤维丛 (fibre bundle)**：底空间 + 每点上的纤维。PDE 将“点”重写为底坐标 + 纤维上的虚体积。
- **黎曼曲面 / 多值结构**：复平面上的多叶结构。与 PDE 的递归虚面、多层级解释相呼应。

---

#### 物理 / 量子

- **希尔伯特空间 / 波函数**：态空间与概率幅。PDE 的虚体积、相位与“第二次坍缩”与之类比。
- **量子概率（幅、密度矩阵）**：非交换观测量、干涉。PDE 用其扩展经典概率在零测集与意义层上的处理。

---

以上概念在具体接口中的使用方式，见各接口文件的 **「向其它领域的输出课题」** 小节。
